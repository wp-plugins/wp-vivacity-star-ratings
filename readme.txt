The Wp Star Rating plugin  most used review plugin on the web. Wp Star Rating allows users to give star rating on the web. User-friendly ratings for: posts, pages, products , events.

== Description ==


WP Star Rating allows you to set up rating and review system for posts, pages and products in your blog. You can set many options for displaying the rating stars displaying top ratings and other statistics generated by the plugin. Plugin includes advanced settings panels that will allow you to control many aspects of rating. Plugin supports use of cache plugins and dynamic loading of data using ajax.

WP Star Rating Plugin with few features.

= Features: =
* 3 shapes are available in 5 colors to fit most variety of themes
* Allows to adjust rating between 3-10
* Turn on/off display of amount of votes
* Display before or after content
* Allows to select for what types of content rating will be displayed
* Allows to disable rating for specific entry
* Choose alignment of the rating
* Works with custom posts types
* Allow or disallow guests to vote
* Allow user to insert feedback regarding the plugin
* Supports Multiple Languages

== Installation ==

1. Upload the WP Star Rating Plugin folder to the /wp-content/plugins/ directory
2. Activate the WP Star Rating plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to 'WP Star Rating' option under 'Dashboard' menu

== Frequently Asked Questions ==

**Can I reset votes?**

Yes, you can by pressing "Reset votes" button on WP Star Rating setting page. 
Note: This only resets votes. Plugin will automatically completely remove its data upon deletion.

**If I don't want to add rating for specific post or page, can I disable it for that specific post or page?**

You certainly can. You just have to tick the box "Disable rating" when editing/publishing post or page.

**Are custom post types supported?**

Yes, custom post types are supported.
